Data Download Information for Safe To Eat Portal

The information on this page is provided to help end-users better understand data downloaded from the Data and Trends page from the State Waterboard's Safe To Eat Portal. If you have questions or comments, please contact cristina@sfei.org.

The data available in the portal have been compiled across a range of studies. They include:

  • Toxic Substances Monitoring Program RWQCB6 Fish 2002
  • RWB6 Donner Lake Fish Bioaccumulation Status
  • RWB7 Fish Tissue Study 2004
  • RWB2 Lakes Sportfish Contamination Study 2004-2005
  • Fish Mercury Project 2005-2007
  • Statewide Lakes Sportfish Contamination Study 2007-2008
  • Statewide Coast Sportfish Contamination Study 2009-2010
  • Statewide Rivers Sportfish Contamination Study 2011-2012
  • 2009 Regional Monitoring Program Status & Trends

New studies will be added to the Portal as they are made available.

Downloaded data exist in a .csv file format. This file format can be opened with Microsoft Excel or other spreadsheet software. Here is a list of all the columns in the data download file and what they represent.

  • CompositeID: A unique identifier for the sample analyzed. Records from the standardized size model have Not Applicable (NA) in this column as they are calculated results.
  • StationCode: A unique identifier for the sampling location.
  • StationName: The name of the sampling location. Note that two different studies may use identically named sampling locations, but will have different StationCodes.
  • CommonName: The common name of the fish sampled.
  • TissueCode: The type of tissue analyzed. Possible values are: FIL = fillet; WHL = whole; WNHTG = whole no head, tail or guts; LIV = liver
  • PrepName: How the tissue was prepared for analysis.
  • SampleCategory: A value representing whether this is an individual or composite sample. Possible values: Individual = 1 fish; Composite = multiple fish; SuperComposite = a composite of composite samples.
  • NumberFishInComposite: The number of fish in the sample.
  • AnalyteGroup: The name of the group to which the contaminant analyzed belongs.
  • AnalyteName: The name of the contaminant analyzed.
  • Result: The concentration of the contaminant analyzed. A blank result reflects a coelution and the result is reported with the coeluting parameter. A 0 represents a non-detect for organics. Inorganic non-detects are represented by half the MDL value.
  • SummedResult: The summed concentration of all analytes in the AnalyteGroup.
  • Unit: The units and basis in which the concentration is reported.
  • NumberOfCongeners: The number of individual contaminants comprising the SummedResult.
  • ResQualCode: Any qualifiers applied to the result.
  • MDL: The method detection limit of the contaminant analyzed.
  • SampleYear: The year in which the sample was collected.
  • ProgramCode: The program code of the study.
  • ProgramName: The program name of the study.
  • ParentProjectCode: The code of the parent project of the study.
  • ParentProjectName: The name of the parent project of the study.
  • ProjectCode: The project code of the study.
  • ProjectName: The project name of the study.
  • TotalLengthAvg: The average total length of the fish in the sample. If sample is an individual, this value is the total length of the fish.
  • TotalLengthUnit: The units in which the total length average is reported.
  • Comment: Comments pertaining to the sample.
  • Latitude: The latitude of the sampling location. Sometimes this is not a precise value, but a generalized one to represent the location of the waterbody.
  • Longitude: The longitude of the sampling location. Sometimes this is not a precise value, but a generalized one to represent the location of the waterbody.
  • Datum: The datum of the latitude/longitude.
